# Hands-on-Labs for Advanced Cybersecurity

Practical laboratory exercises complementing the **Certified Ethical Hacker (CEH) Preparation Guide: Lesson-Based Review of Ethical Hacking and Penetration Testing** book by Sheikh, published by Apress.

## About This Project

This repository contains hands-on lab exercises designed to reinforce concepts from the CEH preparation guide. The labs cover various ethical hacking and penetration testing techniques in a controlled, educational environment.

## Book Information

- **Title**: Certified Ethical Hacker (CEH) Preparation Guide: Lesson-Based Review of Ethical Hacking and Penetration Testing
- **Author**: Sheikh
- **Publisher**: Apress
- **Edition**: 2026

## Lab Structure

```
weeks/
├── week1/
├── week2/
└── ...
```

Each week's folder contains corresponding lab exercises and materials.

## Prerequisites

- Python 3.8+
- JavaScript (Node.js 14+)
- A virtual machine or isolated environment (recommended for security labs)

## Installation

```bash
# Clone the repository
git clone https://github.com/sam-l-112/Hands-on-Labs-for-Advanced-Cybersecurity.git

# Navigate to the project directory
cd Hands-on-Labs-for-Advanced-Cybersecurity
```

## Usage

Refer to the individual lab folders for specific instructions for each exercise.
