# Week04 DarkSword-Lite CTF 報告 - 截圖說明

本說明文件提供學生在撰寫週04報告時所需截圖的範例與擷取建議。
報告中「漏洞現象截圖（至少 3 張）」部分需要包含以下三種類型的截圖：

1. Console 噴 OOB 錯誤
2. Canvas 畫壞（紅色）
3. alert 跳出 flag

以下分別提供每種截圖的具體擷取時機、應呈現內容與備註。

**一般擷取建議**：
- 使用瀏覽器開發者工具 (F12) 或作業系統截圖功擷取。
- 建議儲存為 PNG 格式，檔名簡潔且具描述性（例如：screenshot_console_oob.png）。
- 在報告中以 Markdown 語法嵌入圖片：`![說明](screenshot_console_oob.png)`
- 若圖片過大，可適當調整寬度：`![說明](screenshot_console_oob.png =500x)`
